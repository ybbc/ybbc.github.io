2017 苹果企业开发者账号详细申请流程

Veröffentlicht am 2017-03-09 |

苹果开发者账号分3种，分别是个人账号，公司账号和企业账号。下面是这3者的区别：

类型 App Store 上架 最大uuid数量 费用
个人 是 100 $99
公司 是 100 $99
企业 否 不限制 $299

企业账号一般是公司规模在500人以上的企业，用于内部测试发布的账号，并且该账号发布的应用不能发布在appstore上。
需要注意的是，这种账号一旦到期了，手机上已经安装的APP会无法启动，因此账号的按期续费非常重要。
企业账号的设备数没有限制，并且不用经过App Store的审核，但是仅仅用于内部测试，不可以公开下载，我是帮朋友申请的账号，至于他是用来干什么我也不知道，还是警告一下大家，苹果的管控是非常严格的，任何违背苹果条款使用企业账号，都会有企业账号被封的风险，封号之后使用该证书的APP将会闪退。

申请前准备

  • 公司域名邮箱(必须是公司域名，不然申请不了)
  • 邓白氏编码 D-U-N-S Number(有申请流程)
  • 公司资料(公司英文名、公司网址、法人资料等等)

申请流程

如果不想看文字可以直接预览看图片

1 Apple ID 申请

Apple ID 申请链接
注意：必须是企业域名的邮箱，不然后面不能往下走
密保问题一定要记住，修改资料或者密码都要用到
国家选择中国，除非你公司在其他国家

2 注册登陆后点击Account, 点击Join the Apple Developer Program

留意到左上角是 Apple Developer Program 此时点击Enroll申请的是个人或公司类型的账号

翻到最底部，点击 Enterprise Program 的 learn more

可以看到，现在的左上角变成了 Apple Developer Enterprise Program，中间多了一个 Enterprise，这个时候就是申请的企业类型账号。
一定要多注意 Enterprise，每次点击下一步前都看看是不是 Enterprise，因为企业类型和公司类型账号的申请流程基本差不多，一不小心申请错了就gg了。

可以看到 299 USD，in-house等字眼，可以肯定这就是企业类型账号

点击 start

到了选择类型页面，前面是一些关于Apple ID 的信息，Country 要是China，不然就只能重新申请Apple ID 了。
类型选择 Company / Organization
如果你有了 邓白氏编码 (D-U-N-S Number) 直接点击下一步继续申请

3 邓白氏编码 (D-U-N-S Number) 的申请

点击 Check now 到申请页面

填写公司资料，这里的公司英文名一定要规范，具体规则自己google

填写你的联系信息，填写的邮箱是邓白氏审核结果的邮箱

ok申请完成

马上会收到邓白氏的邮件申请反馈，说会在10天内审核完成，其实基本上2，3天就会有邮件反馈审核结果的

第二天便收到了邓白氏的邮件，填写一下公司资料，联系人信息，公司营业执照等等，填写好了直接回复过去

第三天又收到了邓白氏的邮件，恩，申请通过了，D-U-N-S Number 到手

回到之前的页面，有了 D-U-N-S Number 之后不用管Check now，翻到最下面直接下一步
如果你的Apple ID 不是企业域名的邮箱，那么就会出现以下页面，告诉你企业账号的 Apple ID 必须是企业域名的邮箱

4 企业账号资料的填写

Apple ID 没问题就会到这一步了，这个时候注意些左上角有没有 Enterprise，因为公司账号类型的申请不需要企业域名
选择权限，一般选择第二个，除非你就是老板，选择第二个需要多填写一些资料

填写授权人信息

填写公司资料，D-U-N-S Number

地址没问题直接打钩，Submit

好了，申请已经搞定了，接下来就等待苹果的审核，一般1，2天就会收到苹果的反馈，注意查收邮件

整个申请流程就这么简单，这里列出特别需要注意的几点

  • 1 注意左上角的 Enterprise
  • 2 企业域名命名的邮箱
  • 3 企业英文名要规范
  • 4 企业资料要正确，企业的网址和企业要有关系

ps：公司账号的申请流程基本和企业账号差不多的，附上公司账号的申请流程，仅图片
